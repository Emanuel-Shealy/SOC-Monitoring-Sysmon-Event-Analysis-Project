# SOC-Monitoring-Sysmon-Event-Analysis-Project
Developed a Windows-based security monitoring and alerting environment using Sysmon and PowerShell to simulate SOC-style monitoring operations and real-time event detection workflows.

Technologies Used:
PowerShell
Sysmon
Windows Event Viewer
Windows Security Logs

Features:
Real-time PowerShell activity detection
Command Prompt execution monitoring
Network connection event monitoring
File creation event tracking
Automated alert generation
Event log analysis
Security Events Monitored
Event Type	Detection Method
PowerShell Execution	Sysmon Event Analysis
CMD Execution	Process Monitoring
Network Connections	Sysmon Event ID 3
File Creation	Sysmon Event ID 11

Example Alerts:
[ALERT] PowerShell activity detected!
[NETWORK] Network connection detected!
[FILE EVENT] File creation detected!

Security Monitoring
