# SOC Monitoring - Sysmon Event Analysis Project

## Overview
Developed a Windows-based security monitoring and alerting environment using Sysmon and PowerShell to simulate SOC-style monitoring operations and real-time event detection workflows.

## Technologies Used
- Sysmon
- PowerShell
- Windows Event Viewer
- Windows Security Logs

## Monitoring Capabilities
- PowerShell activity detection
- Command Prompt execution monitoring
- Network connection monitoring
- File creation event tracking
- Event log analysis
- Automated alert generation

## Sysmon Event IDs Monitored

| Event ID | Description |
|---|---|
| 1 | Process Creation |
| 3 | Network Connection |
| 11 | File Creation |
| 22 | DNS Query |

## Alerts Generated
- PowerShell execution alerts
- CMD execution alerts
- Network activity alerts
- File creation alerts
